## Marketing & Public

### Landing Page
```
Design a [product type] landing page. Hero section with [headline], subheadline,
[CTA type] button, [social proof element]. Feature section with [N] columns
showing [feature details]. Final CTA banner. [Color/theme description].
```

### Pricing Page
```
Design a pricing page with [N] tiers: [tier names]. Each card shows price,
feature list with checkmarks, CTA button. [Highlighted tier] has "Most Popular" badge.
```
